# 議員でありながらネット上で一般人を脅します  

![一般人を脅します](images/ogp-004-01.png)

twitter でう〜ちゃんさんが『小坪さんよ、添田氏発言取り消したいってよ。』と事実を述べたにも関わらず、小坪氏は『うそ』と決めつけ口汚い言葉で、う〜ちゃんさんを執拗に脅し続けました。  
小坪氏は行橋市の市会議員です。  
社会的責任のある立場です。  
この様な振る舞いは決して許されません。
